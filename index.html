Code Examples (100% HTML-Free)

Simple Counter Component


  {
  "type": "div",
  "children": [
    { "type": "h2", "text": "{{data.count}} clicks" },
    { "type": "button", "text": "Increment", "onClick": "increment" }
  ],
  "data": { "count": 0 },
  "methods": { "increment": "data.count += 1" }
}

Todo List (with Loop)


  {
  "type": "div",
  "children": [
    {
      "type": "div",
      "style": { "display": "flex", "gap": "8px", "marginBottom": "16px" },
      "children": [
        { "type": "input", "placeholder": "Add a task", "bind": "data.newTodo" },
        { "type": "button", "text": "Add", "onClick": "addTodo" }
      ]
    },
    {
      "type": "ul",
      "children": [
        {
          "type": "li",
          "text": "{{item.text}}",
          "for": "item, idx in data.todos",
          "children": [
            { "type": "button", "text": "Delete", "onClick": "deleteTodo(idx)" }
          ]
        }
      ]
    }
  ],
  "data": { "todos": [], "newTodo": "" },
  "methods": {
    "addTodo": "if (data.newTodo) { data.todos.push({ text: data.newTodo }); data.newTodo = ''; }",
    "deleteTodo": "data.todos.splice(idx, 1)"
  }
}

Real-time Chat Component


  {
  "type": "div",
  "children": [
    {
      "type": "div",
      "style": { "height": "300px", "overflowY": "auto", "border": "1px solid #ddd", "padding": "16px" },
      "children": [
        {
          "type": "div",
          "text": "{{msg.user}}: {{msg.text}}",
          "for": "msg in data.messages"
        }
      ]
    },
    {
      "type": "div",
      "style": { "display": "flex", "gap": "8px", "marginTop": "8px" },
      "children": [
        { "type": "input", "placeholder": "Type a message...", "bind": "data.newMessage" },
        { "type": "button", "text": "Send", "onClick": "sendMessage" }
      ]
    }
  ],
  "data": { "messages": [], "newMessage": "" },
  "methods": {
    "sendMessage": "if (data.newMessage) { data.messages.push({ user: 'You', text: data.newMessage }); data.newMessage = ''; }"
  }
}

Why Choose JSON App Builder?

The Most Minimal Code, Ever.

JSON App Builder is the most minimalistic web framework you will find. You describe your UI and logic in pure JSON and JavaScript—no HTML, no build tools, no complex configuration. If you want to spend less time on boilerplate and more time on ideas, this is for you.

Perfect for prototypes, MVPs, and small apps

Unlike other frameworks that require deep knowledge of their ecosystem, JSON App Builder lets you build working apps with less code and less friction. You can go from idea to working prototype in minutes.

How does it compare?

React, Vue, Angular, and even low-code tools all require more setup, more syntax, and more files. Here, you just write a single JSON object and a few lines of JavaScript for logic. No virtual DOM, no JSX, no templates—just data.

Limitations: This approach is intentionally minimal. While you get authentication, data storage, and real-time updates out of the box, there are limits to what you can customize. For large-scale or highly custom apps, a traditional framework may be better.
